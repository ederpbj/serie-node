## [API NodeJS + Express + Mongo] Estrutura e cadastro | Diego Fernandes

https://www.youtube.com/watch?v=BN_8bCfVp88&t=304s

git clone https://github.com/ederpbj/serie-node.git

> Iniciar

    npm init -y

> Pacotes

    yarn add express
    yarn add body-parser
    yarn add mongoose
    npm install bcrypt

> Start inicial

    node src/index.js
